Git_Github_TP_partie2 est un TP fait pour le cours d' Openclassrooms : "Gérez votre code avec Git et Github"

Pour me servir de support à ce TP, j'ai pris un projet perso que j'ai fais : ce projet avait comme but de reproduire le site d'APPLE, et d'y ajouter un menu burger pour les supports mobiles.


Le but de ce TP, est de créer un repository GIT sur ma machine, de créer plusieurs fichiers, de faire plusieurs commit, et ensuite , après avoir créer un repository sur github, de récupérer mon dépot en local, sur Github.
